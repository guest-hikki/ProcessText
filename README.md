### 使用方法

> `ctrl + s`：保存当前数据
>
> `ctrl + z`：rollback到之前保存的数据
>
> `ctrl + /`：唤出字符选择窗口

0. 选择字符：可以使用热键F12唤出，唤出后重新导入文件即可生效
