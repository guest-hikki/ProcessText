# ProcessText V1.0

### 打开方式

命令行执行执行：`java -Dfile.encoding=utf-8 -jar ProcessText.jar`

或者直接添加环境变量

### 使用方法

1. 导入文件
2. 选择查找模式
3. 输入待查找的字符串进行查找
4. 对需要操作的目标添加标记
5. 对添加标记的文本进行操作
6. 导出数据
